## detected.* block — full schema

The introspector populates this block. Every field is optional; templates MUST always pair `detected.*` lookups with `| default("...")`.

```yaml
detected:
  python:
    auth_dep:                # most common auth dependency in sampled routers (e.g. require_tier_or_guardian, get_current_user, login_required)
    api_prefix_base:         # most common URL prefix in sampled routers (e.g. /api, /v1)
    test_async_pattern:      # decorator used by sampled async tests (e.g. @pytest.mark.asyncio)
    _provenance:
      auth_dep_source:       # path:line of the file where auth_dep was extracted
      api_prefix_base_source:
      test_async_pattern_source:
  swift:
    api_client_class:        # name of the API client class used in sampled Views (e.g. HedgeAPI, APIClient)
    biometric_policy:        # LocalAuthentication policy used in sampled biometric calls (e.g. deviceOwnerAuthenticationWithBiometrics)
    _provenance:
      api_client_class_source:
      biometric_policy_source:
  typescript:
    trpc_router_builder:     # e.g. createTRPCRouter, publicProcedure
    procedure_pattern:       # e.g. publicProcedure.input(...).query(...)
    _provenance:
      trpc_router_builder_source:
```

## Variable scope contract

| Top-level | Source | Notes |
|-----------|--------|-------|
| `{{framework.*}}` | `getConfig().framework` | type, primary, router, orm, ui, languages.* |
| `{{paths.*}}` | `getConfig().paths` | source, aliases.*, web_source, python_source, swift_source, rust_source, ios_source |
| `{{detected.*}}` | `getConfig().detected ?? {}` | Detector-owned per-language conventions. Always paired with `\| default("...")`. |
| `{{config.*}}` | `getConfig()` (full object) | Use only for top-level keys: toolPrefix, python, conventions. Avoid for nested `framework.*` / `paths.*` / `detected.*` (use those forms above). |

All paths are dot-walked via `Object.hasOwn`-only traversal — no prototype walk, no recursive expansion. See spec §"Semantics" rules 2 and 6.
